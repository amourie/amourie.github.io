Random Quotes Generator - infinite wisdom of Anthony Bourdain

I've chosen Anthony Bourdain firstly because I found a site of 30 entertaining quotes by him - bonus! It's a mini site really, but the crux of the code is an array of JS quote objects. I'm randomly selecting a quote from the array to display to the user when the user clicks a button to see additiional quotes by Mr. Anthony Bourdain. I keep track of each quote shown to ensure it is not repeated in the same set of quotes. When all quotes have been used up, the array is reset and the process starts again.

ZMOnline.com, Flava.co.nz, TheHits.co.nz, Hauraki.co.nz

This project encompassed several new responsive site builds for several of NZME's radio stations. We started with the flagship Youth brand / radio station - zmonline.com as part of a small team. My contribution was mostly front-end development, but also some back-end development and CMS integration for certain componenets. SASS themes and the build of CMS functionality, to allow the site to be skinned with different colours and themes to better complement client creative. Content editors were able to to change out not only themes but specify which brand logos the site could use to complement the background skin / theme: for example a glitter logo, hot pink logo etc.

Using the same front-end code and back-end framework as for zmonline.com, we re-launched www.thehits.co.nz, www.flava.co.nz, www.hauraki.co.nz and I contributed in a similar capacity as with ZM dividing my time equally between deveopment tasks and project management of these websites
