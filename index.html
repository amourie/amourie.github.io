Random Quotes Generator - infinite wisdom of Anthony Bourdain

I've chosen Anthony Bourdain firstly because I found a site of 30 entertaining quotes by him - bonus! It's a mini site really, but the crux of the code is a an array of JS quote objects and I'm randomly selecting a quote from the array to display to the user when the user clicks the button to see additiional quotes by Anthony Bourdain. I keep track of each quote shown to ensure it is not repeated in the same set of quotes. When all quotes have been used up, the array is reset and the process starts again.
